PHÚC IELTS — Website hoàn chỉnh, độc lập.

CÁCH ĐƯA LÊN MẠNG (khuyên dùng Netlify kéo–thả — dễ và nhận file lớn):
1. Vào https://app.netlify.com → đăng nhập.
2. Add new site → Deploy manually.
3. Kéo–thả TOÀN BỘ thư mục "phucielts-site" này vào ô đó.
4. Netlify cho bạn một link web chạy được ngay.

Ghi chú:
- Web cần internet để tải React/Babel (unpkg) và Google Fonts — bình thường với mọi web online.
- Video buổi học vẫn nhúng từ YouTube (kênh cũ). Muốn độc lập hẳn thì tải video về và up lên kênh mới.
- Chức năng nộp bài trỏ tới Google Apps Script mới của bạn (đã cấu hình trong app/data.jsx).
